## Project Overview

This React web app visualizes global IQ scores, government education investment (% GDP) and access to education on an interactive world map. Users can explore countries, view average IQ scores or education spending, and focus on specific regions using the sidebar.  

The app is designed to **highlight disparities in access to education** and promote awareness about global educational opportunities.

> The IQ data used in this project comes from publicly available GEMINI datasets. These figures are **not comprehensive** and **do not reflect individual intelligence**. IQ is influenced by numerous cultural, social, and economic factors, and this map is **intended solely as an educational visualization tool**.  
>
> This project does **not rank countries or individuals** and should **not be used to make judgments or policy decisions**. Its purpose is to illustrate **differences in access to education and investment around the world**.
